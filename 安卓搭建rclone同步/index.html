安卓搭建rclone同步

11 Apr, 2025

主要目的是为了平替Foldersync，需要用到Termux和alist

因为FolderSync的webdav协议只支持https，但是alist只支持http，设置https也挺麻烦的，所以就没有去弄。

首先你需要安装一个Termux，然后你要配置一个Alist

Termux换源之后，直接pkg install alist，安装alist，之后的内容就和电脑版差不多，可以参考着来。

别名

别名的设置，alist server如果感觉难记，那就写个别名

方法是创建名称为 .bashrc 的文件，然后在文件里面添加

1alias a1='alist server'

其中a1就是别名，直接在命令行输入a1就可以运行alist server

使用source .bashrc来启用这个命令，source后面跟的是.bashrc的位置。

可以用mt管理器来玩termux，会直接显示termux视角下的文件系统，相当于图像化管理了。

rclone

然后是rclone的配置，同样是pkg install rclone，安装rclone。

然后是配置

直接rclone config 进入配置，然后选择新建配置的对应字母。

配置需要设置 配置名称：自己设置，比如我设置成aaa url：http://127.0.0.1:5244/dav/baidu_123(dav后填写的是你自己想挂载的路径) 用户名：alist中设置的 密码：alist中设置的

别的高级设置 head什么的都可以不填直接空住，点回车（选择默认的选项）。

常用命令

1rclone ls aaa:/

其中aaa选择你自己设置的名称，这个命令主要是用于列出文件夹下的所有文件（包括子文件夹），如果能列出来说明配置成功了。

同步

可以选择sync命令同步，这个是单向同步，bisync是双向同步但是我没搞懂那个。

1rclone sync aaa:/music/华语经典热歌榜200首 /media/

注意这个同步主要是从第一个地址同步到第二个地址。

在Termux中，地址是 storage/shared/你从文件夹打开的默认地址

bug

  1. 如果发现rclone ls 命令列出的东西不对，那先从网页版登录一下alist，注意用火狐，必应，或者谷歌浏览器，国内的夸克什么的可能登陆不了。

登录alist之后，加载到对应挂载的地方然后点旁边的刷新按钮。 如果依然解决不了，关闭alist后重启alist
