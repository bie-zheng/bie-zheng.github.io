hugo配置github page遇到的问题

15 Apr, 2025

用hugo server -b https://bie-zheng.github.io/ --disableFastRender 设置完

并且在hugo.toml中，baseURL = "https://bie-zheng.github.io/"

这俩都设置完，发现bear blog这个主题的别的都能用了，blog还是不能用

发现问题是blog的连接本来应该是 https://bie-zheng.github.io/博客一

测试的时候是 https://bie-zheng.github.io/博客一

搞不清楚了，怒，后面用了vscode的搜索替换功能，把所有的https://bie-zheng.github.io替换成 https://bie-zheng.github.io，然后就没问题了。
